## 💡 Tips for Creating Templates

### 1. Mix Activity Types
Gunakan berbagai type untuk variasi:
```typescript
templates: [
  { text: '...', type: 'PLAYING' },
  { text: '...', type: 'WATCHING' },
  { text: '...', type: 'LISTENING' },
  { text: '...', type: 'COMPETING' }
]
```

### 2. Balance Dynamic & Static
```typescript
templates: [
  { text: '{total} tasks', dynamic: true, type: 'WATCHING' },
  { text: 'task manager', dynamic: false, type: 'PLAYING' }
]
```

### 3. Use Emojis Wisely
```typescript
templates: [
  { text: '🎯 {total} targets', dynamic: true, type: 'WATCHING' },
  { text: '⚡ power mode', dynamic: false, type: 'COMPETING' }
]
```

### 4. Keep It Short
- Max 30 characters recommended
- Discord truncates long text
- Short = more impactful

### 5. Test Empty State
Pastikan template tetap bagus saat {total} = 0:
```typescript
{ text: '{total} tasks active', dynamic: true }
// Output: "0 tasks active" ← Still OK!
```
